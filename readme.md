# alt:V Multiplayer | MLO Streaming

Put all MLO Files in the Stream-Folder.
<br>
Change the Foldername & add the Foldername in your `server.cfg`

```
burgershot/
|-> stream/
|    |-> _manifest.ymf
|    |-> *.ytyp
|    |-> *.ytd
|    |-> *.ydr
|    |-> *.ybn
|    |-> *.ydd
|    \-> *.ymap
|-> resource.cfg
\-> stream.cfg
```
